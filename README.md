# Meu Portfólio

Este projeto é o meu portfólio pessoal, com informações sobre mim, meus projetos e tecnologias.

Link do portfólio: https://portfolio-bryan.example.com

## Publicação no GitHub Pages

O deploy está configurado com GitHub Actions em [deploy-pages.yml](.github/workflows/deploy-pages.yml).

1. Faça push para a branch `main`.
2. No GitHub, acesse `Settings > Pages`.
3. Em `Build and deployment`, selecione `GitHub Actions`.

Depois disso, cada push na `main` publica automaticamente o site.
